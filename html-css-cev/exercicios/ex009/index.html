Listas em HTML5

Listas Ordenadas

Listas Ordenadas são utilizadas quando os itens contidos nela possuem relevância na ordem.
Para criar uma Lista Ordenada em HTML5, se utiliza a tag <ol> e para cada item se deve utilizar <li>.

Passo-a-Passo de como Tomar Água

  1. Pegue um recepiente, pode ser um copo ou garrafa limpa
  2. Encha o recipiente com água
  3. Hidrate-se

Parabéns! Agora, sabemos como tomar água (Já lavou sua garrafa hoje?).

Listas Não Ordenadas

Nas Listas não Ordenadas, como o próprio nome revela, os itens não possuem uma ordem definida.
Para desenvolver essa lista, usamos a tag <ul>.

Listas Mistas

É possível misturar listas! Sabia dessa? Observe:

Hobbies Legais Para Fazer:

  1. Sozinho
    • Pintura
    • Leitura
    • Ouvir uma Playlist
  2. Em Grupo
    • Colagem
    • Noite de Filmes
    • Passeio de Bicicleta

Não precisa seguir à risca! Pode fazer o hobbie que quiser e como quiser, tá bem?
Aliás, para misturar essas listas, basta aninhar as tags dentro de outra.

Listas de Definição

Existem também as listas de definição. Lembre-se do dicionário, é algo bem semelhante.
Para isso, vamos utilizar a tag <dl> para demarcar a lista, <dt> para o termo e <dd> para a definição.
