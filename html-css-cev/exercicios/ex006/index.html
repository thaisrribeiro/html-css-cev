HTML5: A HTML Semântica

A principal diferença entre a HTML4 e HTML5 é no quesito semântica, isso é, significado.
Com isso, diversas tags se tornaram obsoletas, ou seja, não se deve utilizá-las nos documentos HTML atuais.
Então, como saber qual das tags ainda podem ser utilizadas? O site principal responsável por isso é o W3.org, onde é possível verificar o que pode e não pode ser utilizado.

Algumas Tags Obsoletas

<center>, <big>, <marquee>
