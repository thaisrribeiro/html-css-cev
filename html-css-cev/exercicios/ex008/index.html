Código-Fonte

Você sabe como escrever "Hello, World!" em C? É fácil! Observe:


    #include <stdio.h>
    #include <stdlib.h>

    int main() {
        printf ("Hello, World!");
        return 0;
    }

Simples! Não é? Ah... Sobre as tags? Claro!
<pre> vai manter toda a formatação e <code> formata o texto com fonte monoespaçada.
