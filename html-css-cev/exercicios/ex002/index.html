Parágrafos e Caracteres Especiais

Quebra de Linha em Parágrafos

Todo o conteúdo dentro de um parágrafo, por mais que tenha espaços com a tecla enter, o texto será mantido na mesma linha, pois ele está dentro de uma tag <p>.

Caso seja necessário dar um espaçamento dentro de uma tag de parágrafo, é possível utilizar a tag <br> para quebra a linha.
No entanto, deve-se evitar o uso encadeado dessa tag para formar um grande espaçamento. Isso é trabalho das CSS.

Utilizando Símbolos e Emojis

É possível copiar e colar caracteres especiais diretamente no documento HTML. Porém, nem todos os navegadores possuem suporte, então a melhor maneira de utilizá-los é por códigos.

Os símbolos são chamados de HTML Entities. Você pode consultar uma lista delas buscando por "Misc Symbols". Para adicionar ao código, utilize "&#x" e o código correspondente do símbolo. ⚞ ☆ ⚟

Os emojis também são adicionados da mesma forma. 💐🐝
