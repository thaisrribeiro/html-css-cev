Links Externos e Internos

Links são de extrema relevância em sites, pois permitem a navegação tanto externa quanto interna. O que seria da web sem os links?
Bem, para criar links em um documento HTML, utilizamos a tag <a> e o parâmetro href="", apenas substituindo o valor pela URL desejada.

Link Externo

Vamos para outro site? Que tal o YouTube? Há todos os tipos de conteúdo que você pode ver por lá! Clique aqui para acessar.
Há alguns parâmetros que foram utilizados dentro dessa tag:

  • target="", nesse caso, com o valor _blank, para abrir uma nova guia;
  • rel="", preenchido por external para indicar um site externo.

Observe como ficaria se não houvesse esses parâmetros. Clique aqui!
Como é um hyperlink para um site externo, muito provável que a pessoa não volte mais para o nosso site!

Link Interno

Vamos para outra página? Digo, internamente agora!

Ir para segunda página.

Para páginas internas, devemos utilizar os seguintes valores para os parâmetros target="" e rel="", respectivamente:

  • "_self" para indicar que é interno;
  • "next", indicando ser a próxima página;
  • "prev", para a página anterior.

IMPORTANTE! Quando se tem pastas, usamos o caminho delas, por exemplo: "outros/pag3.html" e, para voltar, utilizamos "../" e o nome do arquivo desejado. Usuários de Linux devem estar familiarizados com isso.
